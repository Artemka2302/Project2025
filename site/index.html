BMW и ИИ

Компания BMW активно использует искусственный интеллект для оптимизации HR-процессов. Современные технологии на основе AI делают подбор, адаптацию и обучение персонала более точными и индивидуально ориентированными. С помощью умных чат-ботов и аналитических систем HR-специалисты BMW могут оперативно выявлять перспективных кандидатов, точнее формулировать требования к вакансиям и принимать решения, подкреплённые аналитикой. Этот инновационный подход укрепляет цифровую культуру компании и подтверждает её лидерство в области технологических преобразований.

ИИ для HR: Новый стандарт BMW

BMW использует искусственный интеллект не только для автоматизации рутинных процессов, но и для более точного анализа потребностей персонала. Благодаря обработке больших данных компания выявляет эффективные HR-стратегии, прогнозирует риски текучести и разрабатывает персональные планы развития для сотрудников. Это делает кадровые процессы в BMW более адаптивными, прозрачными и человеко-ориентированными.

Аннотация проекта

Наш проект посвящен исследованию и внедрению искусственного интеллекта в HR-процессы. В ходе практики в компании BMW мы создали инновационный чат-бот, который с помощью AI-технологий помогает HR-специалистам автоматически генерировать тексты вакансий.

Актуальность и проблематика проекта

Актуальность проекта заключается в необходимости автоматизации отдельных HR-процессов в дилерских центрах «BMW» посредством внедрения технологий искусственного интеллекта (далее – ИИ).

Применение ИИ позволит освободить сотрудников от рутинных задач, повысить эффективность управленческих решений, внедрить персонализированный подход к кадрам и усилить имидж компании как технологически продвинутого работодателя. Это, в свою очередь, поспособствует ускоренному привлечению квалифицированных специалистов и повышению их вовлеченности и лояльности.

Проблематика проекта связана с недостаточной эффективностью текущих HR-процессов, ограниченными ресурсами для персонализированной работы с сотрудниками и отсутствием интеграции современных цифровых решений в кадровую практику.

Задачи

  • Исследование современных инструментов и практик применения искусственного интеллекта в сфере управления персоналом.
  • Создание плана и структуры чат-бота с учётом задач HR-отдела и удобства для пользователей.
  • Проведение пользовательского тестирования, сбор обратной связи и внесение необходимых улучшений в систему.
  • Интеграция разработанного решения в повседневную деятельность HR-отдела и оценка его эффективности.

Техническая реализация

Проект реализован в виде Telegram-бота, использующего языковую модель GigaChat для генерации текстов вакансий. Бот взаимодействует с пользователем через диалог, собирая необходимую информацию о вакансии.

Исходный код бота

import { Agent } from "node:https";
import { Telegraf, session } from "telegraf";
import { Stage, WizardScene } from "telegraf/scenes";
import { message } from "telegraf/filters";
import GigaChat from "gigachat";

const bot = new Telegraf("7970149447:AAH5dyN_8FqYWC_qTPnzCDOLVEX5coZYoiQ");

const giga = new GigaChat({
    timeout: 600,
    model: "GigaChat-Max",
    credentials: "NzYyMjk1N2QtNTU5OS00MTk2LTgxMWQtOWI2ZWRkMzg3NGY4OmUzZjgzMDUxLTE4ZjEtNDUyMi04OGQ3LTZlZDU4ZDk1N2M3MA==",
    httpsAgent: new Agent({ rejectUnauthorized: false })
})

const generatePrompt = ({name, education, experience, schedule, place}) => `Напиши вакансию для дилерского центра BMW. В общую часть включи: О компании (часть российской сети BMW; фокус на инновациях, премиум-качество, клиентский опыт). Мы предлагаем (конкурентная зарплата + бонусы, обучение по стандартам BMW, карьерный рост, льготы на авто/страхование, современное оборудование, дружный коллектив). Общие требования (клиентоориентированность, стремление к развитию, надежность, соблюдение стандартов BMW, презентабельный вид). Стиль: мотивирующий, профессиональный, без клише. Акцент на экспертизе, инновациях и премиум-сервисе. Убрать разъяснение терминов. Требования указаны ниже. Название вакансии:  ${name} Образование:  ${education} Опыт работа:  ${experience} График работы: ${schedule} Место работы: ${place} `;

const vacancy = new WizardScene("VacancyScene",
  (ctx) => {
    ctx.reply("Как будет называться вакансия?")
    ctx.wizard.next()
  },
  (ctx) => {
    const response = ctx.message.text
    ctx.wizard.state.name = response

    ctx.reply("Какое образование необходимо для вакансии?")
    ctx.wizard.next()
  },
  (ctx) => {
    const response = ctx.message.text
    ctx.wizard.state.education = response
    
    ctx.reply("Какой опыт работы необходим для вакансии?")
    ctx.wizard.next()
  },
  (ctx) => {
    const response = ctx.message.text
    ctx.wizard.state.experience = response
    
    ctx.reply("Какой график работы будет у этой вакансии?")
    ctx.wizard.next()
  },
  (ctx) => {
    const response = ctx.message.text
    ctx.wizard.state.schedule = response
    
    ctx.reply("Где будет работать сотрудник этой вакансии?")
    ctx.wizard.next()
  },
  async (ctx) => {
    const response = ctx.message.text
    ctx.wizard.state.place = response
    
    ctx.session.messages = [{
      role: "user",
      content: generatePrompt({ ...ctx.wizard.state })
    }]
    
    const responce = await giga.chat({ messages: ctx.session.messages })
    const text = responce.choices[0].message.content

    await ctx.reply(text)
    ctx.session.messages.push({role: "assistant", content: text})

    ctx.scene.leave()
  },
)

const scenes = new Stage([vacancy])
bot.use(session())
bot.use(scenes.middleware())

bot.command("vacancy", Stage.enter("VacancyScene"))

bot.on(message("text"), async (ctx) => {
  const message = ctx.message.text

  ctx.session.messages.push({ role: "user", content: message })

  const responce = await giga.chat({ messages: ctx.session.messages })

  ctx.session.messages.push({ role: "assistant", content: responce.choices[0].message.content})

  await ctx.reply(responce.choices[0].message.content)
  console.log(responce.usage.total_tokens)
})

bot.launch()

Как это работает?

Бот использует пошаговый диалог для сбора информации о вакансии:

  1. Название вакансии
  2. Требуемое образование
  3. Необходимый опыт работы
  4. График работы
  5. Место работы

После сбора всей информации, бот генерирует профессиональный текст вакансии с использованием языковой модели GigaChat, учитывая все стандарты и требования BMW.
